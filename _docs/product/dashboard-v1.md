# MoneyMind – Dashboard V1

## 🎯 Goal
Provide a clear financial overview and guide the user to the next best action.

## 🧩 Core Blocks

1. Wealth Snapshot
- Net worth
- Direct capital
- Locked capital

2. AI Insight
- What stands out
- Why it matters
- Think about

3. Next Step
- Dynamic CTA based on user data
- Links to next tool in flow

4. Financial Maintenance
- Last updated per tool
- Status indicators (OK / Attention / Missing)

5. Progress
- Financial Clarity Level
- Optional percentile (later)

6. Premium Teaser (placeholder)
- Locked advanced insight
- Non-intrusive

## ⚙️ UX Rules
- Mobile-first
- Max 1–2 lines per block
- Clear hierarchy: Insight → Action
- No clutter

## 🔗 Dependencies
- Tool outputs (Roast, Capital Map, etc.)
- AI insight layer
