## Core Purpose

MoneyMind is a **financial intelligence platform** that helps people make better financial decisions so that sustainable wealth becomes a logical outcome.

It is NOT:

- a broker
- a hype-driven finance app
- an affiliate funnel

It focuses on:
- clarity
- structure
- financial understanding
- behavioral awareness
- decision-making
