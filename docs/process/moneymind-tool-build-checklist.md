# MoneyMind Tool Build Checklist

Use this checklist when creating a new MoneyMind tool.

The goal is to keep every tool consistent, clean, and aligned with the MoneyMind platform design and architecture.

---

# 1. Create the tool folder

Create a new directory inside the project:

/tool-name/

Example:

/leakage/
/cost-of-delay/
/builder/

Add:

index.html

---

# 2. Copy the starter template

Copy the template:

/docs/templates/moneymind-tool-starter.html

Rename it to:

index.html

inside the new tool folder.

---

# 3. Update basic tool information

Change:

Title
